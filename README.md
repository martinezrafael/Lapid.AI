# Lapid.AI
API REST assíncrona orientada a eventos para refino inteligente de notas e automação de carreira. Desenvolvida em Node.js, TypeScript e PostgreSQL, utilizando mensageria nativa com RabbitMQ (Exchange/Topology) para gerir concorrência, prefetch (throttling) e isolamento de falhas por IA com Circuit Breakers e DLQ.
